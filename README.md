\# 🌐 AI Translator



A multilingual AI assistant that accepts input in any language and provides AI-powered responses in English.



\## Features

\- Type in any language or transliterated text

\- Automatically translates to English

\- AI powered responses using Groq API

\- Simple and clean web interface



\## Technologies Used

\- Python

\- Flask

\- Groq API (LLaMA 3.3)

\- Google Translate (googletrans)



\## How to Run

1\. Clone the repository

2\. Install dependencies: pip install -r requirements.txt

3\. Create a .env file and add your GROQ\_API\_KEY

4\. Run: python app.py

5\. Open browser and go to https://ai-translator-1-ji8b.onrender.com/



\## Example

\- Input: "jai shree ram"

\- Translated: "Jai Shri Ram"

\- AI Response: "Jai Shri Ram! That's a traditional Hindu greeting..."
